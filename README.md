# Kairo Landing Page

The public landing page for Kairo, built with Next.js, React, and TypeScript.

## Development

```bash
npm install
npm run dev
```

The local site runs at `http://localhost:3000`.

To run a production build locally:

```bash
npm run build
npm run start
```

## Verify

```bash
npm test
npm run typecheck
npm run build
```

The waitlist is a local preview. It does not send or store email addresses.
