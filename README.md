# Kairo Landing Page

The public landing page for Kairo, built with React, TypeScript, and Vite.

## Development

```bash
npm install
npm run dev
```

The local site runs at `http://127.0.0.1:5273`.

## Verify

```bash
npm test
npm run typecheck
npm run build
```

The waitlist is a local preview. It does not send or store email addresses.
